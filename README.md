# Accident-Detection-Emergency-Alert-System
Automatic accident detection system that differentiates between hard braking and crashes, sending emergency alerts only for serious accidents with a 15-second cancellable countdown.
Features

Real-time acceleration monitoring (MPU6050)
Distinguishes hard braking from crashes
15-second countdown with manual cancellation
Emergency SMS with GPS coordinates
Visual indicators (Green/Yellow/Red LEDs)
Audio alarm system

Components

Arduino Uno
MPU6050 (Accelerometer/Gyroscope)
Buzzer
3x LEDs (Green, Yellow, Red)
Push Button (Cancel)
